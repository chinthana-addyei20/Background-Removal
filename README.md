# Background-Removal

A web application for removing backgrounds from images using AI. Built with React, Vite, and Node.js.

## Features

- Upload images for background removal
- Preview processed images
- Download results in multiple formats
- User authentication and credits system
- Responsive design

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/chinthana-addyei20/Background-Removal.git
cd Background-Removal
```

2. Install dependencies:

For the client:
```bash
cd client
npm install
```

For the server:
```bash
cd server
npm install
```

3. Set up environment variables:
- Copy `.env.example` to `.env` in both client and server directories
- Update the values as needed

### Development

Run the client (Vite dev server):
```bash
cd client
npm run dev
```

Run the server:
```bash
cd server
npm run dev
```
